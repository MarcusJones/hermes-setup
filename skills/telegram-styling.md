# Telegram Styling & Formatting

This skill defines the visual identity for Hermes when communicating via Telegram. Use these rules to ensure clarity, readability, and a high-end technical aesthetic on mobile and desktop clients.

## Core Aesthetic
- **Dividers**: Use `━` (U+2501) to separate sections.
- **Lists**: Use `•` (bullet points) for primary items and standard indentation for nested items.
- **Headers**: Use **Bold Labels** followed by a divider or a newline. Do not use `#` headers.

## Layout Templates

### 1. General Summary
```
Section Title
━
• Item one
• Item two
```

### 2. Task/Todo List
```
[Status] Category Name
━
• [ ] Task description
• [x] Completed task
```

### 3. Technical Data
- Always use triple backticks ` ``` ` for logs, code, or terminal output to enable one-tap copying on mobile.

## Media Delivery
- Telegram supports native display for:
  - `.png`, `.jpg`, `.webp` (Photos)
  - `.ogg` (Voice bubbles)
  - `.mp4` (Videos)
  - `.pdf` (Documents)

## Guidelines
- Keep paragraphs short (2-3 sentences) for mobile readability.
- Use emojis sparingly and only as functional icons (e.g., 🔴 for urgent, 💼 for work).
